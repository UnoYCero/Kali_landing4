⚠️ Funciones de Métricas Requeridas

No se pudieron consultar todos los recursos de almacenamiento porque tu base de datos no tiene instaladas las funciones auxiliares de PostgreSQL. Por favor, ejecuta la siguiente instrucción SQL en el **SQL Editor** de tu consola de Supabase: 

-- 1. Tamaño de la Base de Datos
create or replace function get_database_size()
returns bigint language sql security definer as $$
  select pg_database_size(current_database());
$$;

-- 2. Tamaño de Almacenamiento (Storage Files)
create or replace function get_storage_size()
returns bigint language sql security definer as $$
  select coalesce(sum((metadata->>'size')::bigint), 0)::bigint from storage.objects;
$$;

-- 3. Conteo de Usuarios Activos (MAU)
create or replace function get_monthly_active_users()
returns bigint language sql security definer as $$
  select count(*)::bigint from auth.users;
$$;

Una vez ejecutada la consulta, refresca esta página para ver las métricas de tu cuenta.
